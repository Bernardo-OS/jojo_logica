# Logic Bomb Game
##Assignment for the mathematical logic subject applied to computing

**There is a bomb that need to be desarmed cuting off some wires of its circuit**

- You need to be carefull to cut the correct wires in the correct order and fast enough to desarm before the counting down set to 0

- There are 6 wires identified by its colors:
<span style="color: red;">Red (R)</span>, <span style="color: blue;">Blue (B)</span>, <span style="color: green;">Green (G)</span>, <span style="color: yellow;">Yellow (Y)</span>, <span style="color: white;">White (W)</span>, <span style="color: black;">Black (K)</span>

- You have a scheme for the disarmed bomb state expressed as true logic propositions as a clue to desarm, where a positive value indicates that the wire is disconnected


*The program runs right in the terminal. Accepting input by the string of the letter corresponding to the colors of the wires*

*Python vertion 3.14.4*
